# Project 5 Neighborhood Map

Instructions
------------
Open the index.html file in a browser.

Contents
--------
The js folder contains the app.js file, jquery file, offline file and knockout file. The css folder contains the style.css file, offline files, and bootstrap file. The img folder contains the Foursquare image. The main directory contains the favicon.ico, index file and README.
